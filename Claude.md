   - Two-column layout (70/30 split)

   - Package logo displayed
   - Uses fragment and incremental animations

7. **What I Learned Along the Way**
   - **Technical Skills:**
     - R package development workflows
     - Documentation best practices (roxygen2, pkgdown)

     - Git and GitHub collaboration
   - **Soft Skills:**
     - Communication with diverse teams
